# LoadingSpinner

The `LoadingSpinner` component displays a loading indicator with optional message and size variants.

## Usage

```tsx
import { LoadingSpinner } from '@/components/common';

<LoadingSpinner size="lg" message="Loading..." />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Size of the spinner |
| `message` | `string` | - | Optional message to display |
| `fullPage` | `boolean` | `false` | Whether to show as full page loader |
| `overlay` | `boolean` | `false` | Whether to show with dark overlay |

## Examples

### Basic Usage

```tsx
<LoadingSpinner />
```

### With Message

```tsx
<LoadingSpinner message="Loading data..." />
```

### Full Page Loader

```tsx
<LoadingSpinner fullPage message="Loading application..." />
```

### With Overlay

```tsx
<LoadingSpinner overlay message="Saving changes..." />
```

## Size Variants

| Size | Description |
|------|-------------|
| `sm` | Small spinner (24px) |
| `md` | Medium spinner (default, 40px) |
| `lg` | Large spinner (56px) |
| `xl` | Extra large spinner (80px) |

## Accessibility

- The component includes `role="status"` and `aria-live="polite"` for screen readers
- Loading state is announced to assistive technologies
- Messages are provided in `aria-label` where appropriate
