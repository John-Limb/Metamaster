# Metamaster Repository

Metamaster is an advanced media management platform built for organizing, tracking, and optimizing your media library. Designed with a rich feature set and intuitive UI, it supports both movies and TV shows, offering seamless integration with third-party services and deep extensibility.

---

## Features

- **Effortless Media Management:** Intelligently catalog your library with automated metadata.
- **Extensive File Analysis:** File attribute extractions with FFmpeg.
- **Multi-layered Cache:** OMDB and TVDB rate-limits avoided via Redis.
