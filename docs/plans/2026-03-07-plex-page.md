# Plex Now Playing Page Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a dedicated `/plex` page showing live Plex playback sessions, with a sidebar nav item and a not-connected fallback.

**Architecture:** A thin `PlexPage` wraps the existing `NowPlayingPanel` component (from `2026-03-07-plex-now-playing.md`). The page reads connection state from `usePlexStore` to show a "not connected" fallback — no new data fetching. Route added to `App.tsx` following the existing lazy-load pattern. `SiPlex` icon added to sidebar nav.

**Tech Stack:** React, React Router, `usePlexStore` (Zustand), `NowPlayingPanel` (TanStack Query, polls every 15s), `SiPlex` from `react-icons/si`.

**Prerequisites:** Complete Tasks 1–5 from `docs/plans/2026-03-07-plex-now-playing.md` **but in Task 5, skip the step that wires `NowPlayingPanel` into `SystemHealthPage`** — that wiring belongs here instead.

**Step 3: Create `frontend/src/pages/PlexPage.tsx`**

```tsx
import React, { useEffect } from 'react'
import { Link } from 'react-router-dom'
import { usePlexStore } from '@/stores/plexStore'
import { NowPlayingPanel } from '@/components/features/plex/NowPlayingPanel'

export function PlexPage() {
  const { connection, isLoading, fetchConnection } = usePlexStore()

  useEffect(() => {
    fetchConnection()
  }, [fetchConnection])

  if (isLoading) {
    return (
      <div className="space-y-6" data-testid="plex-page-skeleton">
        <div className="h-8 w-24 bg-slate-200 dark:bg-slate-700 rounded animate-pulse" />
        <div className="h-32 bg-slate-200 dark:bg-slate-700 rounded-xl animate-pulse" />
      </div>
    )
  }

  return (
    <div className="space-y-6">
      <div>
        <h1 className="text-2xl font-bold text-slate-900 dark:text-white">Plex</h1>
        <p className="text-slate-500 dark:text-slate-400 mt-1">
          Live playback sessions · updates every 15s
        </p>
      </div>

      {connection ? (
        <NowPlayingPanel />
      ) : (
        <div className="text-sm text-slate-500 dark:text-slate-400">
          Plex is not connected.{' '}
          <Link to="/settings" className="text-primary-600 dark:text-primary-400 hover:underline">
            Configure it in Settings.
          </Link>
        </div>
      )}
    </div>
  )
}

export default PlexPage
```
